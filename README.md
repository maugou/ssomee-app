## 소미마켓

### **주요기능**

- 상품 목록
  - 검색 기능
  - 필터(정렬)기능
  - 초기화 기능
- 상세 페이지
  - 구매 및 장바구니 담기 기능
- 장바구니
  - 상품 확인 및 금액 표시
  - 구매 기능
- 구매내역

### **개발 환경**

- React Native 0.64
- React Navigation v5
- 상태관리 : Redux
